# chaos_crypt

基于混沌理论的高性能加密演示 APP

***


**项目名称：** ChaosCrypt Mobile (基于混沌理论的高性能加密演示 APP)

**角色设定：**
你现在的身份是一名精通 Flutter 跨平台开发和 C++ 底层性能优化的资深架构师。我们需要开发一款移动端 APP，用于展示一套基于混沌理论（Chaos Theory）的 C++ 加密算法的安全性与高性能。

**技术栈要求：**

1. **前端框架：** Flutter (Dart)。
2. **核心算法：** C++ (已有的 `.cpp` 代码，包含 OpenMP 并行计算逻辑)。
3. **通信桥接：** 使用 `dart:ffi` (Foreign Function Interface) 实现 Dart 与 C++ 的高性能互通，避免拷贝大量内存。
4. **平台目标：** Android 和 iOS。

**核心设计理念：**

* **科技感与数学美：** 界面需要体现“混沌”、“随机”、“加密”的视觉隐喻。UI 设计请自由发挥，推荐使用高科技、深色模式或赛博朋克风格，强调算法的硬核属性。
* **交互直觉化：** 这是一个技术展示应用，操作必须简单直接，避免繁琐的文件管理。

APP由三个模块组成,分别是 可视化加密,文件加密,加密性能三个模块,可以用底部的三个tab来切换,其中可视化可以放到最中心,其是最主要的模块

需要注意的是,底层的加密功能先占位,先不实现,先实现UI和交互逻辑


**功能模块详细描述：**

#### 1. 模块一：可视化加密

这是用户体验的核心入口，用于直观展示“加密前”与“加密后”的差异。

* **输入方式：**
* **即时拍摄：** 调用系统相机拍照。
* **相册选取：** 从图库选择图片。
* **内置样本：** 提供 "Lena" 图或 "Mandrill" 图作为标准测试源。


* **处理逻辑：** 图片数据通过 FFI 传入 C++ 层，进行混沌置乱与扩散，返回加密后的像素数据。
* **展示要求 (UI 自由发挥)：**
* 必须同屏或交互式展示 **“明文图”** 和 **“密文图（噪点图）”** 的对比。
* **数据可视化：** 必须包含 **R/G/B 直方图 (Histogram)** 的对比展示。
* *逻辑说明：* 明文图的直方图应波峰波谷明显；密文图的直方图应呈现完美的“均匀分布 (Uniform Distribution)”，以此证明算法的安全性。

* **输出：** 提供“分享”按钮，目的是微信,方式是分享文件的方式。


#### 2. 模块二：文件加密模块

实现加密文件的社交分享与解密。

* **文件格式：** 自定义后缀 `.lzu`。
* **加密流程：** 文件选择选择任意人家，将其加密为 `.lzu` 文件（完整数据加密），通过系统 Share Sheet 分享给微信/QQ好友。
* **解密流程：**
* APP 需要注册系统级的文件关联 (Android Intent-filter / iOS Document Types)。
* 当用户在微信中点击 `.lzu` 文件并选择本 APP 打开时，APP 自动启动并导航至解密页面。
* 自动调用 C++ 解密算法，还原并展示原始文件,原始文件存放到根目录的chaos_crypt文件夹下。

#### 3. 模块三：极限性能仪表盘 (Speed Benchmark)

用于展示 C++ 核心算法结合 OpenMP 后的极致处理速率，解决手机传输大文件不便的问题。

* **数据源：** **不需要读取本地文件**。在内存中虚拟生成指定大小的随机数据流（10MB / 100MB / 500MB / 1GB）。
* **交互逻辑：** 用户选择虚拟文件大小，点击“开始测试”。
* **展示要求 (UI 自由发挥)：**
* 动态展示加密过程中的实时速率 (MB/s)。
* 展示总耗时。
* 如果有条件，可以设计一个图表对比“单线程”与“多线程(OMP)”的性能差异（可选）。
* 视觉上需要体现“极速”、“算力全开”的动态效果。
